# Smart Redirector
Chrome extension to automatically redirect to archive.is for content blocked by paywalls (i.e. _NY Times_, _Wall Street Journal_, etc)

## Local Development

### Install
```sh
yarn install
```

### Build
```sh
yarn run build
```
